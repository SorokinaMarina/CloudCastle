# Тестовое задание для компании "Cloud Castle"

## Требования:
### Привести существующую вёрстку к макетному виду, редактируя css и js файлы.

Допускается незначительное изменение html файла. Карточку "summary" нужно сверстать полностью.

Интерактив: карточки кликабельны, одновременно можно выбрать не более четырёх карточек. Если выбрано четыре, то оставшиеся переводятся в состояние disabled. Карточка "summary" не имеет интерактивности.

Код должен быть валидным и кроссбраузерным (Chrome, Edge, Firefox, Yandex Browser, Safari, Safari iOS) с поддержкой корректного отображения при любых разрешениях.
